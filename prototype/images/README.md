# Images — drop your photos here

I could **not** pull these from Facebook automatically — Facebook blocks
unauthenticated access (it serves a login wall / a default silhouette instead
of the real images). So please add the files manually:

1. Open your Facebook/Instagram page **while logged in**.
2. Right-click each photo → **Save image as…** (or download from Meta Business Suite → Content).
3. Save them here with these **exact file names**, and the site will use them automatically:

| File name | Where it shows | Best size |
|---|---|---|
| `hero.jpg` | Full-width hero background (top of page) | wide/landscape, ~1920×1080, dark works best |
| `about.jpg` | "About Us" section photo | ~1200×1000 |

(JPG or PNG is fine — if you use PNG, rename to `.jpg` anyway **or** tell me and I'll switch the references.)

### Want a photo gallery / before-after section too?
Add files named `gallery-1.jpg`, `gallery-2.jpg`, … and tell me how many —
I'll build a gallery section and wire them in.

### The logo
If you have the logo file, save it as `logo.png` (transparent background ideal)
and I'll replace the "ADC" text mark in the header & footer with it.

> Until files are added, the site gracefully shows the styled gradient
> placeholders — nothing looks broken.
