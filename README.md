# Auto Drive Center — Website

Landing page for **Auto Drive Center** (أوتو درايف سنتر), an engineering-led car
maintenance workshop in El-Salam, managed by Eng. Mina Matta.

## Live preview

- **English:** [`/prototype/index.html`](prototype/index.html)
- **Arabic (RTL):** [`/prototype/index-ar.html`](prototype/index-ar.html)

> Once GitHub Pages is enabled, the client can view it at the published URL
> (the root redirects to the English page).

## Project structure

| Path | Purpose |
|---|---|
| `prototype/index.html` | English homepage (design prototype) |
| `prototype/index-ar.html` | Arabic / RTL homepage |
| `prototype/styles.css` | Design system (charcoal / red / white) |
| `prototype/styles-rtl.css` | RTL overrides for the Arabic page |
| `prototype/script.js` | Sticky nav, scroll reveals, counters, form |
| `ELEMENTOR-BUILD-GUIDE.md` | Step-by-step rebuild in WordPress + Elementor |
| `auto-drive-center-homepage.html` | Native Gutenberg block markup (first draft) |
| `index.html` | Root redirect → `prototype/index.html` |

## Production plan

The prototype is the **design spec**. The production site is rebuilt in
**WordPress + Elementor on Hostinger** so the client can edit text, phone
numbers, and services visually with no code. See `ELEMENTOR-BUILD-GUIDE.md`.
